# سوالات متداول (FAQ) TEOS

## عمومی

### ۱. TEOS چیست؟
TEOS یک سیستم عامل سازمانی مبتنی بر Telegram است که شامل ماژول‌های موزیک، فروش سرویس، کیف پول، پشتیبانی و پنل مدیریت می‌باشد.

### ۲. آیا استفاده از TEOS رایگان است؟
نسخه Community رایگان است. نسخه Enterprise دارای امکانات پیشرفته‌تر مانند چندمستأجری، پلاگین‌های تخصصی و پشتیبانی優先 است.

### ۳. چه پیش‌نیازهایی برای نصب وجود دارد؟
Docker، Docker Compose، Python 3.12 (برای توسعه)، یک سرور با حداقل 2 هسته و 4GB RAM، و یک دامنه (برای Webhook).

## نصب و راه‌اندازی

### ۴. چگونه TEOS را نصب کنم؟
مراحل در مستندات deployment.md آمده است. به طور خلاصه: `git clone`، تنظیم `.env`، `docker-compose up -d`، `alembic upgrade head`.

### ۵. آیا می‌توانم بدون Docker نصب کنم؟
بله، اما توصیه نمی‌شود. می‌توانید به صورت دستی وابستگی‌ها را نصب کرده و `python -m app.main` را اجرا کنید.

### ۶. چگونه Webhook را تنظیم کنم؟
ابتدا مطمئن شوید سرور شما از طریق HTTPS قابل دسترس است، سپس:

curl -X POST "https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://your-domain.com/webhook"
